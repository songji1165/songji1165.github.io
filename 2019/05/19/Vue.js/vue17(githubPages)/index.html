Vue Cli Github Pages (참고)


1. Github Pages

  1. github에서 제공하는 사이트 호스팅 서비스
    • github repositories의 소스코드를 기반으로 웹사이트 생성
  2. 커스텀 url을 무료 제공
    • 웹사이트 기본 url : https://UserName.github.io
  3. 서버쪽 코드는 제공되지 않음!
