    > **github에 프로젝트 올리기**
        - git init : 새로운 local repository 생성
        - git add : 변경된 파일 local repository에 추가
        - git commit : add한 파일을 local repository에 저장
        - git push : local repository를 remote repository(github repository)에 업로드

        1. 작업할 폴더로 이동한다 ! 
            git 해당파일
        2. git add . 
        3. git commit -m "message"
        4. git push -m
